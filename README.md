# VEONPAD
its a minimalistic macropad with 9 key switches .I made this so its really simple for a beginner to build and under stand the process and make one for them self.from the pcb to the case is simplified enough so anyone can build one without some Einstein brain .. 

 # SPECIFICATION
 The project use a seed xiao rp2040 microcontroller paired  with cherry mx key switches.It uses 4N4148 diode to prevent ghosting .


 # pcb
 its a simple 9 key matrix .it only uses 6 gpio pins of the microcontroller instead of 9.So you can add rotary encoder screen etc if you want to make one yourself.
